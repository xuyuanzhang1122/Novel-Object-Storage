面向运维与代理的自托管对象存储

Novel Object Storage

面向脚本、CLI 和 AI 自动化的轻量对象存储。上传、检索、元数据维护、公开链接和 API key 管理都在同一个工作台里完成。

模式 CLI 优先 API curl / Bearer / X-Api-Key
存储 磁盘 + JSON 无外部数据库依赖
快速开始 
cp .env.example .env
npm install
npm start
